HOW IT WORKS

 

One config. Every template.

 

Add a theme.extend block to your config. Every key becomes a Tailwind-style class you can use directly in your markup — no plugins, no build step.

mailc.config.js 
                              export
                              default {
                              theme: {
                              extend: {
                              colors: {
                              brand: {
                              DEFAULT: '#6d28d9',
                              light: '#a78bfa',
                              dark: '#4c1d95',
                              },
                              surface: '#18181b',
                              muted: '#71717a',
                              },
                              fontFamily: {
                              display: "'Helvetica Neue', Arial, sans-serif",
                              },
                              borderRadius: {
                              card: '12px',
                              },
                              },
                              },
                              };

WHAT YOU GET

 

Everything your design system needs.

🎨
Colors
Flat keys, nested palettes, or both. <code>bg-brand</code>, <code>bg-brand-light</code>, <code>text-surface</code> — all resolved automatically.
🔤
Typography
Custom font stacks via <code>fontFamily</code>. Applied globally with <code>mc-attributes</code> or per-component as a class.
📐
Spacing & Sizing
Custom steps in your spacing scale. Use <code>p-card</code>, <code>gap-section</code> — any key you name.
⬡
Border Radius
Name your corner sizes. <code>rounded-card</code> becomes <code>border-radius:12px</code> everywhere.
🔒
rem → px auto-conversion
Paste Tailwind's default theme as-is. mailc converts every <code>rem</code> value to <code>px</code> at build time — email clients require it.
♿
Accessibility baked in
Enable <code>accessibility.enabled</code> to get automatic <code>lang</code> attribute injection, missing-alt warnings, and WCAG contrast checks.
