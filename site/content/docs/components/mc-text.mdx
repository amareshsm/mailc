Rich text content block. Accepts HTML inside.

## Quick facts

| | |
|---|---|
| Category | `content` |
| Allowed parents | `mc-column`, `mc-hero` |
| Allowed children | _(none)_ |
| Accepts text content | yes |
| Accepts `class` attribute | yes |
| Compiles to | `<div>`, `<p>`, `<span>` |

## Required attributes

_(none)_

## Compiles to

Output elements: `<div>`, `<p>`, `<span>`.

Div wrapper preserves block layout; p and span for inline text formatting.

## Example markup

```html
<mc-text>Rich text content block</mc-text>
```
