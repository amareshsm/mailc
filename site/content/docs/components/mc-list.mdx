Ordered or unordered list. Children must be mc-list-item.

## Quick facts

| | |
|---|---|
| Category | `content` |
| Allowed parents | `mc-column`, `mc-hero` |
| Allowed children | `mc-list-item` |
| Accepts text content | no |
| Accepts `class` attribute | yes |
| Compiles to | `<table>`, `<tr>`, `<td>`, `<ul>`, `<ol>`, `<li>` |

## Required attributes

_(none)_

## Compiles to

Output elements: `<table>`, `<tr>`, `<td>`, `<ul>`, `<ol>`, `<li>`.

Outer table normalises Outlook list rendering; ul/ol/li for semantic markup.

## Example markup

```html
<mc-list></mc-list>
```
