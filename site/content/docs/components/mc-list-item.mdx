Single list item inside mc-list. Accepts inline HTML.

## Quick facts

| | |
|---|---|
| Category | `content` |
| Allowed parents | `mc-list` |
| Allowed children | _(none)_ |
| Accepts text content | yes |
| Accepts `class` attribute | yes |
| Compiles to | `<li>` |

## Required attributes

_(none)_


## Optional attributes

| Attribute | Type | Default | Example | Description |
|---|---|---|---|---|
| `class` | `tailwind-classes` | — | `text-gray-900` | Utility classes. |
| `id` | `string` | — | `item-1` | Unique identifier. |
| `color` | `color` | — | `#333333` | Text color. |
| `background-color` | `color` | — | `#f5f5f5` | Item background color. |
| `font-size` | `css-value` | — | `14px` | Font size for this item. Overrides the inherited size from mc-list. |
| `line-height` | `css-value` | — | `1.6` | Line height for this item. Overrides the inherited leading from mc-list. |
| `font-weight` | `normal` \| `bold` \| `bolder` \| `lighter` \| `100` \| `200` \| `300` \| `400` \| `500` \| `600` \| `700` \| `800` \| `900` | — | `bold` | Font weight. |
| `font-style` | `normal` \| `italic` \| `oblique` | — | `italic` | Font style. |
| `text-decoration` | `none` \| `underline` \| `line-through` | — | `none` | Text decoration. |
| `padding` | `css-value` | — | `4px 0` | Padding around the item content. |


## Compiles to

Output elements: `<li>`.

Semantic `<li>` tag; inline styles inherit from mc-list when not overridden.

## Example markup

```html
<mc-list-item>Single list item inside mc-list</mc-list-item>
```

## Programmatic access

```ts
import { introspect } from 'mailc'

const spec = introspect.component('mc-list-item')
```

See [`introspect.component(type)`](/docs/api/introspect#introspect-component) for the full `ComponentSpec` shape.
