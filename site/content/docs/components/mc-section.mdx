A full-width horizontal row. Holds one or more mc-column or mc-group children.

## Quick facts

| | |
|---|---|
| Category | `container` |
| Allowed parents | `mc-body` |
| Allowed children | `mc-column`, `mc-group`, `mc-raw` |
| Accepts text content | no |
| Accepts `class` attribute | yes |
| Compiles to | `<table>`, `<tr>`, `<td>` |

## Required attributes

_(none)_

## Compiles to

Output elements: `<table>`, `<tr>`, `<td>`.

Full-width outer table plus Outlook conditional inner table for column layout.
