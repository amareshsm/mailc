Head section. Contains meta, styles, and global attribute defaults.

## Quick facts

| | |
|---|---|
| Category | `head` |
| Allowed parents | _(none)_ |
| Allowed children | `mc-attributes`, `mc-preview`, `mc-style`, `mc-title` |
| Accepts text content | no |
| Accepts `class` attribute | yes |
| Compiles to | `<head>`, `<style>`, `<meta>` |

## Required attributes

_(none)_


## Optional attributes

| Attribute | Type | Default | Example | Description |
|---|---|---|---|---|
| `class` | `tailwind-classes` | — | — | Utility classes. |
| `id` | `string` | — | `head` | Unique identifier. |


## Compiles to

Output elements: `<head>`, `<style>`, `<meta>`.

Generates the `<head>` block with meta charset, viewport, and collected styles.

## Example markup

```html
<mc-head></mc-head>
```

## Programmatic access

```ts
import { introspect } from 'mailc'

const spec = introspect.component('mc-head')
```

See [`introspect.component(type)`](/docs/api/introspect#introspect-component) for the full `ComponentSpec` shape.
