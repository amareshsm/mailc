Loop over a collection using a "for each" style.

## Quick facts

| | |
|---|---|
| Category | `logic` |
| Allowed parents | _(none)_ |
| Allowed children | _(none)_ |
| Accepts text content | no |
| Accepts `class` attribute | yes |
| Compiles to |  |

## Required attributes

| Attribute | Type | Default | Example | Description |
|---|---|---|---|---|
| `collection` | `string` | — | `items` | Collection variable name from template data. |
| `as` | `string` | — | `item` | Variable name for each item. |


## Optional attributes

| Attribute | Type | Default | Example | Description |
|---|---|---|---|---|
| `class` | `tailwind-classes` | — | — | Utility classes. |
| `id` | `string` | — | — | Unique identifier. |


## Compiles to

Output elements: .

Compile-time loop — no HTML wrapper emitted.

## Example markup

```html
<mc-for-each collection="items" as="item"></mc-for-each>
```
