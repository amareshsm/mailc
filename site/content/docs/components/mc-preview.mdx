Preheader text — shown in inbox preview. Hidden in rendered email.

## Quick facts

| | |
|---|---|
| Category | `head` |
| Allowed parents | `mc-head` |
| Allowed children | _(none)_ |
| Accepts text content | yes |
| Accepts `class` attribute | yes |
| Compiles to | `<div>` |

## Required attributes

_(none)_


## Optional attributes

| Attribute | Type | Default | Example | Description |
|---|---|---|---|---|
| `class` | `tailwind-classes` | — | — | Utility classes. |
| `id` | `string` | — | `preview` | Unique identifier. |


## Compiles to

Output elements: `<div>`.

Hidden div with overflow:hidden and font-size:0 to suppress visual rendering.

