Clickable button with automatic Outlook VML fallback.

## Quick facts

| | |
|---|---|
| Category | `content` |
| Allowed parents | `mc-column`, `mc-hero` |
| Allowed children | _(none)_ |
| Accepts text content | yes |
| Accepts `class` attribute | yes |
| Compiles to | `<div>`, `<table>`, `<td>`, `<a>`, `<v:roundrect>` |

## Compiles to

Output elements: `<div>`, `<table>`, `<td>`, `<a>`, `<v:roundrect>`.

VML v:roundrect for Outlook rounded buttons, div+a fallback for modern clients.

## Example markup

```html
<mc-button href="https://example.com">Clickable button with automatic Outlook VML fallback</mc-button>
```
