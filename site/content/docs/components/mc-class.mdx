Defines a named attribute bundle applied to components via mc-class="name".

## Quick facts

| | |
|---|---|
| Category | `head` |
| Allowed parents | `mc-attributes` |
| Allowed children | _(none)_ |
| Accepts text content | no |
| Accepts `class` attribute | yes |
| Compiles to |  |

## Required attributes

| Attribute | Type | Default | Example | Description |
|---|---|---|---|---|
| `name` | `string` | — | `hero-heading` | Unique class name referenced with mc-class="..." on components. |

| Attribute | Type | Default | Example | Description |
