A column inside mc-section. Stacks vertically on mobile (or stays side-by-side when wrapped in mc-group).

## Quick facts

| | |
|---|---|
| Category | `container` |
| Allowed parents | `mc-section`, `mc-group` |
| Allowed children | `mc-button`, `mc-divider`, `mc-image`, `mc-list`, `mc-raw`, `mc-spacer`, `mc-table`, `mc-text` |
| Accepts text content | no |
| Accepts `class` attribute | yes |
| Compiles to | `<td>`, `<div>` |

## Required attributes

_(none)_

## Compiles to

Output elements: `<td>`, `<div>`.

td for Outlook fixed-width layout, div with display:inline-block for modern clients.

## Example markup

```html
<mc-column></mc-column>
```
