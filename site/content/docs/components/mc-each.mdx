Loop over an array. Renders children for each item.

|---|---|
| Category | `logic` |

## Required attributes

| Attribute | Type | Default | Example | Description |
|---|---|---|---|---|
| `items` | `string` | — | `products` | Array variable name from template data. |
| `as` | `string` | — | `product` | Variable name for each item. |

| Attribute | Type | Default | Example | Description |

| `id` | `string` | — | — | Unique identifier. |
