mailc compiles `.mc` markup or JSON into email-safe HTML that renders consistently across Gmail, Outlook, Apple Mail, Yahoo, and Samsung Mail.

## Status

mailc is at **v0.1** — the compiler is solid (3,775+ tests passing) but the API surface is still stabilising. Pin to exact versions until 1.0.
